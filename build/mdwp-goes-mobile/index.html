mdwp goes Mobile

20.08.2013

Neuer Service für das mobile Web Mobile Demo Site

Wir haben unser Dienstleistungsangebot um professionelle Services rund um das mobile Web erweitert. Durch den Einsatz einer speziellen Software Lösung können wir Ihnen und den Besuchern Ihrer Websites wunderschöne Ergebnisse liefern.

Die mobilen Sites haben das Look & Feel von Apps und lassen sich wie solche bedienen. Geben Sie sich nicht mit einfachen Responsive Designs zufrieden.

Responsive Design Unter Responsive Design versteht man eine CSS-Technik, die das Layout einer Seite automatisch der Größe des Displays von Smartphones und Tablets anpasst. Ein Nachteil ist, dass gerätespezifische Besonderheiten, wie z.B. ein ‘Click to call’ Button, nur mittels CSS nicht umsetzbar sind.

Mobile Site Eine mobile Site ist häufig als Subdomain wie m.example.com erreichbar. Durch Geräteerkennung wird ein Redirect ausgeführt. Ein Vorteil gegenüber Responsive Designs liegt darin, dass auch die Inhalte angepasst werden können. Diese Inhalte können auch von einem CMS wie Drupal verwaltet werden. Diese Technik setzt mdwp* ein.

Für wen sind extra mobile Sites geeignet?

Grundsätzlich für jeden, der seinen Website Nutzern auf Smartphones einige besondere Funktionen wie click to call, click to text (SMS), Tischreservierungen für Restaurants und eine App ähnliche Oberfläche und Bedienung bieten möchte. Letzteres ist zwar auch mit Responsive Design möglich, aber um einiges aufwendiger. m.sites bieten sich für lokale Geschäfte, Restaurants, Kanzleien, Ärzte und Handwerker besondern an. Denn diese können von den schnellen Kontaktmöglichkeiten via Smartphone besonders profitieren.

Bedeutet eine extra Site erhöhten Pflegeaufwand? Mit der von uns eingesetzten SaaS Lösung pflegen Sie Ihre Inhalte wie bisher z.B. mit dem von uns empfohlenem CMS Drupal. Die mobile Site wird automatisch mit der Drupal Site synchronisiert. Lesen Sie mehr über “Mobiles Business”.
