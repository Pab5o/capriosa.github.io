Professioneller Chat für jede Website

10.12.2011

Mit einer Chat box auf der eigenen Site können Sie die Kommunikation mit den Besuchern Ihrer Site enorm verbessern. Ein Chat wirkt sehr persönlich und kann das Vertrauen Ihrer Kunden fördern. Also ein excellentes Mittel zur Kundenbindung, aber auch zur Online Akquise.

Das amerikanische Unternehmen Olark bietet eine günstige, einfach zu benutzende Chat box für jede Website an. Für den Chat selbst können Sie nahezu jeden Instant Messenger benutzen.

Um das Widget einzusetzen müssen Sie sich bei Olark registrieren und ein JavaScript Snippet einer Seite oder einem Template Ihrer Site hinzufügen. Für Drupal 6 gibt es auch ein Modul für die Integration der Chat box auf Ihrer Drupal Site.

Nach der Einbindung des Snippets müssen Sie noch Ihren Messenger in wenigen Schritten konfigurieren und schon kann es losgehen.

Sie können Olark 14 Tage kostenlos testen. Wenn Sie Olark danach weiterverwenden möchten, können Sie zwischen 4 verschiedenen Paketen, mit unterschiedlichstem Funktionsumfang, auswählen. Die Preise liegen zwischen 14 und 219 € pro Monat.

Welches Paket für Sie in Frage kommt, hängt insbesondere von der Anzahl der Chat Operatoren ab, die den Chat betreuen sollen.

Olark bietet umfangreiche Konfigurationsmöglichkeiten. Diese lassen sich bequem auf der Website von Olark für jede Chat box ändern. So können Sie zwischen unterschiedlichsten Themes wählen und auch schnell hin und her “switchen“. Wir setzen hier gerade das Weihnachts-Theme ein.

Die urspünglich englischen Beschriftungen können ebenso schnell geändert und auch für unterschiedliche Situationen angepasst werden.

So kann der einleitende Text z.B. für eine bestimmte, zeitlich begrenzte Kampagne explizit geändert werden. Oder Sie weisen, so wie wir hier, auf bestimmte Zeiten hin, in denen der Chat betreut wird.

In Zeiten, in denen Sie oder Ihre MitarbeiterInnen nicht online sind, kann das Widget von Ihren Kunden/Besuchern trotzdem benutzt werden, um Ihnen schnell eine kurze Nachricht per E-Mail zu senden. Eine prima alternative zum herkömmlichen Kontaktformular einer Site.

Auf der Konfigurationsseite von Olark finden sich weitere, paketabhängige Anpassungsmöglichkeiten für das Widget.

Insgesamt macht der Service von Olark einen sehr guten Eindruck.
